# Simple React Counter App

A simple React application featuring an interactive counter with dynamic colors.

## Features

- Interactive counter with increment/decrement
- Reset functionality
- Dynamic color changes based on count
- Responsive design
- Modern UI with smooth animations

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Technologies Used

- React 18
- Vite
- CSS3
